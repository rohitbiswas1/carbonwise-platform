# CarbonWise - Carbon Footprint Awareness Platform

A fully interactive carbon footprint calculator and awareness platform built with React, TypeScript, Vite and Tailwind CSS.

## Features
- Interactive CO2 calculator (Transport, Energy, Food, Shopping)
- Visual dashboard with charts and grade system
- Indian carbon offset programs (SankalpTaru, TERI, Grow-Trees, WTI)
- Eco Champions Leaderboard
- Dark mode support
- Pledge certificate generator
- Calculation history tracker
- Fully accessible (WCAG AA)

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Chart.js
- Lucide React Icons
